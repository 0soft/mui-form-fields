# Getting Started

MUI Form Fields was developed by [ZeroSoft](https://zerosoft.com.br/en/) to help
us build lots of forms to our clients with ease. We based ourselves on
[Material UI](https://material-ui.com/) and made the most so we can build a
form with the minimum amount of code possible.

You can make questions, propose improvements and changes on our repository
[here](https://github.com/0soft/mui-form-fields)

## Installation

Install all of MUI Form Fields Peer Dependencies

```bash
npm i --save @date-io/core@^1.3.6 @date-io/moment@^1.3.6 @material-ui/core@^4.1.0 @material-ui/icons@^4.1.0 classnames@^2.2.6 core-js@^3.1.3 css-vendor@^2.0.2 final-form@^4.13.1 is-plain-object@^3.0.0 jss moment@^2.24.0 react@>=16.8.6 react-dom@>=16.8.6 react-dropzone@^10.1.5 react-final-form@^6.0.1 react-select@^3.0.4 react-number-format@^4.0.8
```

Then install MUI Form Fields

```bash
npm i --save mui-form-fields
```

## Usage

This Form Components are primarily based on Final Form, you can take a look
into their [documentation](https://github.com/final-form/final-form) for more
advanced instructions.

import {
  FormDateField,
  FormDialog,
  FormFileUploadField,
  FormPhoneField,
  FormSwitchField,
  FormSelectField,
  FormTextField,
} from '../src';
import Form from "./FormTest";
import ToggleButton from "./ToggleButton";
import { Divider } from '@material-ui/core';
import { Playground } from 'docz';

You'll always need a `<Form>` component wrapping our Form Fields, so as long as
you have a code like this:

<Playground>
  <Form
    render={() => {
      return (
        <FormTextField
          icon="person"
          name="name"
          label="Name"
        />
      );
    }}
  />
</Playground>

This way you may just start importing our components and play around with a
combination of them:

```javascript
import {
  FormDateField,
  FormDialog,
  FormFileUploadField,
  FormPhoneField,
  FormSwitchField,
  FormTextField,
} from 'mui-form-fields';

// Do your magic here...
```

<Playground>

    render={() => {
      return (
        <>

            icon="person"
            name="name"
            label="Name (min: 3 characters)"

            label="Email"
            validate="required | email"
          />

</Playground>

## Examples

Even complex dialogs can be mounted as easily as:

<Playground>
  <ToggleButton
    text="Open Dialog"
    variant="contained"
    color="primary"
    render={(open, setOpen) => {
      // The really important code:
      return (
        <FormDialog
          size="sm"
          title="Create User"
          onClose={() => setOpen(false)}
          onSubmit={async (...args) => {
            return new Promise((resolve) => {
              setTimeout(() => setOpen(false), 2000);
            })
          }}
          open={open}
          dividers
        >
          <FormTextField icon="person" name="name" label="Full Name" />

          <FormPhoneField name="phone" label="Phone" />

          <FormSwitchField icon="security" name="protected" label="Protected" />

          <FormFileUploadField
            icon="attachment"
            name="identification"
            label="Social Security"
          />
        </FormDialog>
      );
    }}
  />
</Playground>
